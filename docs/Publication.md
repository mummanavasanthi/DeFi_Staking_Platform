# GitHub Publication & Contract Details

## Project Title

DeFi Staking Platform

## Project Overview

The DeFi Staking Platform is a decentralized application (DApp) that allows users to connect their MetaMask wallet, stake ETH, calculate rewards, view balances, and withdraw funds through a Solidity smart contract deployed on the blockchain.
