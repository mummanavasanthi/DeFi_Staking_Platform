# Mainnet Deployment

## Project Title

DeFi Staking Platform

## Objective

The objective of deployment is to publish the smart contract and frontend application so users can access the platform through a live blockchain network and web interface.

## Smart Contract Deployment

### Network

SCAI Mainnet

### Deployment Tool

Remix IDE

### Wallet Used

MetaMask

### Smart Contract Address

0x5464C15c05fF36902C66e836232525Af0904c611

### Deployment Process

1. Smart contract written in Solidity.
2. Contract compiled using Remix IDE.
3. MetaMask connected to SCAI Mainnet.
4. Contract deployed successfully.
5. Contract address stored for frontend integration.

## Frontend Deployment

### Platform

Vercel

### Frontend Technology

* React.js
* Vite
* Ethers.js

### Deployment Process

1. Frontend connected to deployed smart contract.
2. Source code pushed to GitHub.
3. GitHub repository connected to Vercel.
4. Build and deployment completed successfully.
5. Live application generated.
